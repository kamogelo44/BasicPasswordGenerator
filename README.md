# Basic Password Generator

A Java program that generates random passwords based on user preferences.

## What This Program Does

- Prompts user for password length
- Asks if user wants numbers and special characters
- Generates a random password with the specified criteria
- Displays the generated password

## How to Run

Make sure you have Java installed
Open the project in your preferred code editor
Locate and run the BasicPasswordGenerator.java file
Follow the instructions on screen

## Sample Usage

Enter password length: 12
Do you want your password to consist of numbers and special characters? (yes/no): yes
Generated Password: k8#mD2pQ9!vL
END OF PROGRAM
